# v0.1.1 山西医科大学 Zotero 论文参考文献样式

本次更新主要修正正文引用和学位论文录入说明。

## 本次更新

- 正文引用调整为上角标数字编号
- 更新学位论文导师录入方式说明
- 统一 README 中的字段填写说明

## 录入约定

- 学位论文导师在 `其他` 中填写：`Editor: 姓 || 名`
- 学位论文类型填写在 `论文类型`
- 专利国别填写在 `Jurisdiction`
- 技术标准代号填写在 `Number`

## 已知限制

- 普通期刊条目不会自动补 DOI 或 URL
- 字体、行距、悬挂缩进和段落格式仍需在 Word 中设置

## English

This update mainly adjusts in-text citations and clarifies thesis data entry rules.

- In-text citations now render as superscript numeric references
- Thesis advisor should be stored in `Extra` as `Editor: Family || Given`
- Patent country should be stored in `Jurisdiction`
- Standard number should be stored in `Number`
