# v0.1.0 山西医科大学 Zotero 论文参考文献样式

山西医科大学学位论文参考文献 Zotero 样式首个公开版本。

## 本次发布

- 提供可直接安装的 Zotero CSL 样式：`shanxi-medical-university-thesis-numeric.csl`
- 支持著作、期刊、电子文献、报纸、学位论文、专利、技术标准和会议论文
- 补充 README、更新日志和发布流程说明

## 录入约定

- 学位论文导师填写在 `Editor`
- 学位论文类型填写在 `Type`
- 专利国别填写在 `Jurisdiction`
- 技术标准代号填写在 `Number`

## 已知限制

- 正文引用当前为普通方括号编号，不是上标
- 字体、行距、悬挂缩进和段落格式仍需在 Word 中设置

## English

First public release of the Zotero CSL style for the Shanxi Medical University thesis bibliography format.

- Style file: `shanxi-medical-university-thesis-numeric.csl`
- Supported items: book, journal article, web page, newspaper article, thesis, patent, standard, and conference paper
- Thesis advisor should be stored in `Editor`
- Thesis type should be stored in `Type`
